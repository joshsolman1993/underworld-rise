# Funkciólista és UI Terv

## 1. Dashboard (Főoldal)
- 
- Felül: Status bar (HP, Energia, Pénz, Idő).
- Bal oldalsáv: Navigáció (Város részei).
- Közép: Aktuális események, logok.

## 2. Helyszínek és Funkciók

### A "Slum" (Kezdő negyed)
- **Sötét sikátor:** Itt lehet alap bűntényeket elkövetni.
- **Edzőterem:** `Willpower` elköltése stat növelésre (Strength, Defense, Agility).

### Városközpont
- **Bank:** Pénz betét/kivét (pénzmosási mechanikával).
- **Pláza (Shop):** Fegyverek, páncélok, ételek vásárlása.
- **Kórház:** Ha HP = 0, itt ébredsz. Fizethetsz a gyorsabb gyógyulásért.
- **Rendőrség/Börtön:** Itt látod, ki van lecsukva. Lehetőség van "kiszabadítani" társakat (magas kockázat) vagy óvadékot fizetni.

### Üzleti Negyed (High Level)
- **Tőzsde:** Részvények vétele/eladása (dinamikus árazás 6 óránként).
- **Ingatlanügynökség:** Saját lakás vásárlása (növeli a `Willpower` regenerációt).
- **Casino:** Blackjack, Roulette (pénzszórási lehetőség).

## 3. Közösségi Funkciók
- **Fórum/Chat:** Globális chat és Banda chat.
- **Piac:** Játékosok egymás közötti kereskedelme (Auction House).
- **Bounty System (Vérdíj):** Pénzt tűzhetsz ki valaki fejére, ha névtelenül akarod elintéztetni.

## 4. Mission System (Küldetések)
- Napi küldetések ("Lopj el 5 autót").
- Történet alapú láncolat, ami feloldja az új városrészeket.
